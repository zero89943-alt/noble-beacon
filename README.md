# Noble Beacon

A quiet public doorway for the IAMRE'KAI / Fe'Kai / Re'Kai living archive.

This is a living archive and civic design laboratory. It is not final canon, not legal authority, not a religion, not a government, and not a loyalty system. Contributions are welcomed as sources, critiques, playtests, designs, and proposals. Nothing enters the archive without review.

## First path

1. Choose a seed.
2. Learn its meaning and status.
3. See one artifact.
4. Perform one small DO.
5. Optionally submit a bounded contribution.

## Core maxims

- Life strengthens Life.
- Shield before sword.
- Stewardship before ownership.
- What remains thriving is the measure.
- The machine carries many voices. The human chooses the song.
