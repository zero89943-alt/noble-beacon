# License Decision Needed

Before public release, choose what is open, what is attribution-required, and what remains reserved.

Suggested separation:

- Public website text: permissive reuse with attribution, if desired.
- Archive canon: source-preserved, not casually forked as authority.
- Software tools: open-source license only after deliberate choice.
- Visual assets: confirm rights and intended reuse limits.

Package tools, not ideology.
