# Agent Workflow

Suggested agents:

- Beacon Steward: prepares public copy.
- Source Ledger Agent: organizes sources and source cards.
- Warding Auditor: checks for hidden power, dark patterns, and panopticon drift.
- Contribution Packet Clerk: turns submissions into provisional packets.
- Moderation Steward: labels and routes, without secret scoring.
- GRM-1 Root Cartographer: tests root alignment.

All agents are advisory, visible, contestable, non-sovereign, and require human approval before public posting or archive integration.


## D2.31 Contribution Button Bridge

The contribute form is local-only in v0.2. Packets are copied or downloaded, then manually uploaded/pasted into ChatGPT for parsing. A future server agent may collect, label, and digest packets, but may not write to the archive or canonize. Re'Kai approval remains mandatory.
